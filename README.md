# 🎮 Balance Game (밸런스 게임 웹사이트)

Flask로 만든 간단한 밸런스 게임 웹 애플리케이션입니다.

---

## 🚀 기능

- 회원가입 / 로그인 / 로그아웃
- 게시글 작성 (밸런스 게임 질문 생성)
- 투표 기능 (A vs B)
- 게시글 수정 / 삭제 (본인만 가능)
- 세션 기반 권한 관리

---

## 🛠 기술 스택

- Python (Flask)
- SQLite3
- HTML / CSS (Jinja2 템플릿)
- Git / GitHub

---

## 🔐 주요 기능 설명

- 로그인한 사용자만 글 작성 가능
- 작성자만 수정/삭제 가능
- 세션(session) 기반 인증 처리
